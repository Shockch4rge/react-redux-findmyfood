# react-redux-findmyfood
A restaurant lookup & review full-stack app.

# Built using:
### Front-End
- React.js library
- Vite bundler with react-ts template
- Material UI framework
- React Redux with Redux Toolkit Query

### Back-End
- express.js
- node.js

### Database
- MySQL

### Miscellaneous
- TypeScript
- node-mysql/mysql2
